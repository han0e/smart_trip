## ✨ Project Overview
이 프로젝트는 2024년 5월, 8인 가족의 오키나와 여행을 위해 제작된 맞춤형 웹 애플리케이션입니다. 복잡한 엑셀 파일이나 종이 일정표 대신, 모바일과 데스크탑에서 최적화된 UI로 실시간 일정을 확인하고 예산을 관리할 수 있도록 설계되었습니다.

## 🚀 Key Features

### 📅 인터랙티브 타임라인 (Schedule)
- 3박 4일간의 상세 일정을 시간순으로 시각화
- **Plan B** 시스템: 날씨나 상황에 따른 대안 일정 즉시 확인 가능
- 장소별 구글 맵(길안내) 다이렉트 링크 연결

### 💰 스마트 예산 관리 (Budget)
- 실시간 환율(929 KRW/100 JPY)을 적용한 원화/엔화 자동 환산
- 항공, 숙소, 렌터카, 입장료 등 카테고리별 비용 정산
- 직관적인 지출 내역 테이블 및 요약 정보 제공

### 📱 사용자 경험 (UX/UI)
- **Modern Design**: Glassmorphism 및 다크 모드/라이트 모드 최적화
- **Interactive Animations**: Framer Motion을 활용한 부드러운 카드 전환 및 인터랙션
- **Responsive**: 모든 기기에서 완벽하게 작동하는 반응형 레이아웃

## 📦 Getting Started

1. **저장소 클론**
   ```bash
   git clone https://github.com/your-username/smart-trip.git
   cd smart-trip
   ```

2. **의존성 설치**
   ```bash
   npm install
   ```

3. **로컬 실행**
   ```bash
   npm run dev
   ```

4. **접속**
   브라우저에서 `http://localhost:3000`을 엽니다.
